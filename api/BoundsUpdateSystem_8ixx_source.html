30 /**
31 * @brief System that updates AABB colliders based on entity transforms.
32 *
33 * @details
34 * This system recalculates the world-space AABB for entities by transforming
35 * the canonical model AABB through the composed local transform (Translation * Rotation * Scale)
36 * and the parent's world transform.
