22 /**
23 * @brief A SpawnStrategy that delegates spawning logic to a callback function.
24 *
25 * @details
26 * CallbackSpawnStrategy provides a lightweight way to define custom spawn
27 * behavior without creating a dedicated subclass. The spawn logic is
28 * encapsulated in a `std::function` passed at construction time.
29 *
30 * Useful for simple or one-off spawn configurations, rapid prototyping,
31 * or when the spawn logic is defined dynamically.
32 *
33 * Example usage:
34 * ```cpp
35 * auto strategy = std::make_unique<CallbackSpawnStrategy>(
36 * [](const GameWorld*, const Level* level, GameObject* obj, const UpdateContext&) {
37 * auto rng = helios::util::Random(1234);
38 * float x = rng.randomFloat(-10.0f, 10.0f);
39 * float y = rng.randomFloat(-10.0f, 10.0f);
40 * obj->get<TransformComponent>()->setTranslation({x, y, 0});
41 * return true;
42 * }
43 * );
44 * ```
45 *
46 * @see SpawnStrategy
47 * @see GameObjectSpawnSystem
48 */
