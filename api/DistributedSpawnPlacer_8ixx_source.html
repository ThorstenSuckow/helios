26 /**
27 * @brief Spawn placer that assigns entities to predefined spawn points.
28 *
29 * @details DistributedSpawnPlacer uses a fixed-size array of locations
30 * to distribute spawned entities. Entities are assigned to locations
31 * based on their cursor position within the spawn batch, dividing
32 * the batch evenly across all available spawn points.
33 *
34 * ## Distribution Algorithm
35 *
36 * Entities are mapped to locations using: `bin = N * (position / spawnCount)`
37 *
38 * This ensures even distribution when spawn count exceeds location count,
39 * and direct mapping when spawn count equals location count.
40 *
41 * ## Use Cases
42 *
43 * - **Fixed spawn points:** Enemies spawning at predefined positions
44 * - **Formation patterns:** Entities placed in specific arrangements
45 * - **Level design:** Designer-defined spawn locations
46 *
47 * ## Usage
48 *
49 * ```cpp
50 * // Create placer with 3 fixed spawn points
51 * auto placer = std::make_unique<DistributedSpawnPlacer<3>>(
52 * helios::math::vec3f{-50.0f, 100.0f, 0.0f},
53 * helios::math::vec3f{ 0.0f, 100.0f, 0.0f},
54 * helios::math::vec3f{ 50.0f, 100.0f, 0.0f}
55 * );
56 *
57 * // 3 entities spawn at the 3 locations
58 * // 6 entities: 2 per location
59 * ```
60 *
61 * @tparam N Number of spawn locations.
62 *
63 * @see SpawnPlacer
64 * @see AxisSpawnPlacer
65 */
