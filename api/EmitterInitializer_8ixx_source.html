21 /**
22 * @brief Initializer that configures spawned entities based on emitter state.

24 * @details EmitterInitializer uses the velocity from the EmitterContext to
25 * set the direction and movement intent of spawned entities. This is commonly
26 * used for projectiles that should inherit direction from the firing entity.

28 * Components affected:

33 * @pre The SpawnContext must contain a valid EmitterContext.

35 * @see SpawnInitializer
36 * @see EmitterContext
