16 /**
17 * @brief Strongly-typed identifier for font families used in text rendering.
18 *
19 * `FontId` provides a compile-time or runtime identifier for font families
20 * registered with the `TextRenderer`. It uses FNV-1a hashing to convert
21 * string names to unique 32-bit identifiers.
22 *
23 * ## Usage
24 *
25 * ```cpp
26 * // Compile-time construction from string literal
27 * constexpr FontId arialFont("Arial");
28 * constexpr FontId monoFont("Monospace");
29 *
30 * // Register font with TextRenderer
31 * textRenderer.addFontFamily(arialFont, "fonts/arial.ttf");
32 *
33 * // Use in TextMesh
34 * TextMesh props{arialFont, {100.0f, 200.0f}, 1.0f};
35 * ```
36 *

38 * @see TextMesh
