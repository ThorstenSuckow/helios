21 /**
22 * @brief Spawn condition that limits spawning based on a maximum rate per second.
23 *
24 * @details MaxPerSecCondition provides rate-limited spawning by calculating a spawn
25 * budget based on elapsed time and a configured maximum spawn rate. This enables
26 * smooth, time-distributed spawning rather than bursts.
27 *
28 * The spawn budget is calculated as: `maxRequestsPerSecond * deltaTime`.
29 *
30 * Example usage:
31 * ```cpp
32 * // Allow up to 10 spawns per second
33 * auto condition = std::make_unique<MaxPerSecCondition>(10);
34 * spawnSystem.setCondition(std::move(condition));
35 * ```
36 *
37 * @see SpawnCondition for the base interface
38 * @see TimerSpawnCondition for interval-based spawning
39 */
