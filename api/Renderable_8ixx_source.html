26 /**
27 * @brief Abstract base class for objects that can be rendered by the rendering system.
28 *
29 * A `Renderable` represents any visual entity that can submit render commands to the
30 * rendering pipeline. Concrete implementations define how the object emits itself
31 * to a `RenderQueue` and how uniform values are written for shader consumption.
32 *
33 * Subclasses must implement:
34 * - `localAABB()`: Returns the local-space axis-aligned bounding box.
35 * - `writeUniformValues()`: Writes object-specific uniform values to a uniform map.
36 * - `emit()`: Submits render commands to the render queue.
37 *
38 * @see helios::rendering::RenderQueue
39 * @see helios::rendering::RenderPrototype
40 */
