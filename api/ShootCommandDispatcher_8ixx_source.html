32 /**
33 * @brief Dispatcher that routes ShootCommands to the ProjectilePoolManager.
34 *
35 * @details ShootCommandDispatcher is a typed command dispatcher that handles
36 * ShootCommand instances targeting specific GameObjects. When a ShootCommand
37 * is dispatched, the dispatcher:
38 *
39 * 1. Retrieves the source entity's position (TranslationStateComponent)
40 * 2. Retrieves the aim direction (Aim2DComponent)
41 * 3. Retrieves the source velocity for inheritance (Move2DComponent)
42 * 4. Creates a ProjectileSpawnRequest with all gathered data
43 * 5. Submits the request to the ProjectilePoolManager for deferred spawning
44 *
45 * This enables the command-based shooting architecture where input systems
46 * emit ShootCommands that are later processed during the manager flush phase.
47 *
48 * @see ShootCommand
49 * @see ProjectilePoolManager
50 * @see ProjectileSpawnRequest
51 */
