20 /**
21 * @brief Command that triggers a shooting action on a GameObject.
22 *
23 * @details ShootCommand is a TargetedCommand that initiates projectile firing
24 * on the target entity. The command carries an intensity value (typically from
25 * trigger pressure or button state) that controls fire rate or projectile behavior.
26 *
27 * When executed, the command:
28 * 1. Retrieves the target's ShootComponent
29 * 2. Optionally retrieves Move2DComponent for velocity inheritance
30 * 3. Calls `ShootComponent::shoot()` with intensity and source velocity
31 *
32 * The source velocity allows projectiles to inherit the shooter's momentum,
33 * preventing projectiles from spawning "behind" a fast-moving entity.
34 *
35 * Example usage:
36 * ```cpp
37 * // In an input system
38 * float triggerValue = inputSnapshot.gamepadState().triggerRight();
39 * if (triggerValue > 0.0f) {
40 * commandBuffer.add<ShootCommand>(player.guid(), triggerValue);
41 * }
42 * ```
43 *
44 * @note The target GameObject must have a ShootComponent attached for this
45 * command to have any effect. Move2DComponent is optional.
46 *
47 * @see ShootComponent
48 * @see TargetedCommand
49 * @see TwinStickInputSystem
50 */
