23 /**
24 * @brief High-level text component that can be attached to game objects.
25 *
26 * `TextRenderable` represents a piece of text that can be rendered in the scene.
27 * It combines a `TextRenderPrototype` (shader and font configuration) with
28 * instance-specific data like the text string and positioning.
29 *
30 * ## Responsibilities
31 *
32 * - Hold the text string to be rendered.
33 * - Reference a shared `TextRenderPrototype` for shader and font settings.
34 * - Store instance-specific `DrawProperties` (position, scale, font).
35 * - Allow property overrides via `TextShaderPropertiesOverride`.
36 *
37 * ## Usage
38 *
39 * ```cpp
40 * auto prototype = std::make_shared<TextRenderPrototype>(shader, textProps);
41 * DrawProperties drawProps{FontId{1}, {100.0f, 200.0f}, 1.0f};
42 *
43 * TextRenderable text("Score: 0", prototype, drawProps);
44 * text.setText("Score: 100"); // Update dynamically
45 * ```
46 *
47 * @see TextRenderPrototype
48 * @see TextRenderCommand
49 * @see DrawProperties
50 */
