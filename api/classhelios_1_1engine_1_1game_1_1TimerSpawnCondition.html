Detailed Description

A SpawnCondition that triggers spawning at fixed time intervals.

TimerSpawnCondition uses a countdown timer to determine when GameObjects should be spawned. Each time the countdown reaches zero, it resets and increments the spawn budget. If the elapsed time exceeds multiple intervals (e.g., during a frame spike), multiple objects will be budgeted for spawning.

Example usage: ```cpp // Spawn one object every 2 seconds auto condition = std::make_unique<TimerSpawnCondition>(2.0f);

auto spawnSystem = std::make_unique<GameObjectSpawnSystem>( std::move(pool), level, std::move(condition), std::move(strategy) ); ```

Calculates the spawn budget based on elapsed time.

Decrements the countdown by the frame's delta time. For each interval that has elapsed, increments the spawn budget by one. This handles frame spikes gracefully by spawning multiple objects if needed.
