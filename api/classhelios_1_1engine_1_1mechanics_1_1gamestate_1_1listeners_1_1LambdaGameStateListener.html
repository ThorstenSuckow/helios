Detailed Description

Lambda-based implementation of GameStateTransitionListener.

Allows registering callbacks for game state transitions without creating a dedicated listener subclass. Useful for simple, inline reactions to state changes.

## Usage

```cpp auto listener = std::make_unique<LambdaGameStateListener>( [](auto& ctx, GameState from) { }, [](auto& ctx, auto transitionCtx) { }, [](auto& ctx, GameState to) { } ); stateManager.addGameStateListener(std::move(listener)); ```
