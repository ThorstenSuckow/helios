Detailed Description

System that synchronizes TransformComponents with SceneNodes.

This system bridges the gameplay simulation and the rendering scene graph. It propagates local transformations from dirty TransformComponents to their corresponding SceneNodes, triggers a scene graph update, and then reads back the computed world transforms into the TransformComponents.
