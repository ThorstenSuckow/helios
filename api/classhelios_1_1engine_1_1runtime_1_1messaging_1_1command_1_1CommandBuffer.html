Detailed Description

Abstract base for command buffers that collect and flush commands.

CommandBuffer defines the interface for deferred command execution. Commands are accumulated during the frame and flushed at defined commit points in the game loop. Concrete implementations (e.g., TypedCommandBuffer) provide type-safe storage and routing.
