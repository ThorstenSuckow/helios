Detailed Description

template<typename CommandType>
class helios::engine::runtime::messaging::command::TypedCommandHandler< CommandType >

Type-safe handler for a specific command type.

TypedCommandHandler provides the interface for receiving commands of a specific type during the TypedCommandBuffer flush. Managers that process commands implement this for each command type they handle.

A single Manager may implement multiple TypedCommandHandler specializations:

```cpp class SpawnManager : public Manager, public TypedCommandHandler<SpawnCommand>, public TypedCommandHandler<DespawnCommand> { bool submit(SpawnCommand cmd) noexcept override { ... } bool submit(DespawnCommand cmd) noexcept override { ... } }; ```

Handlers are registered with the ResourceRegistry during Manager::init() and looked up by the TypedCommandBuffer at flush time.
