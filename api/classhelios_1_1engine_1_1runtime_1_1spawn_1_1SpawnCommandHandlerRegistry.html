Detailed Description

Registry that maps SpawnProfileIds to their SpawnCommandHandlers.

SpawnCommandHandlerRegistry provides lookup from spawn profile IDs to the handlers responsible for processing spawn and despawn commands for those profiles. This enables the spawn/despawn system to route commands to the correct manager without direct coupling.

The registry is owned by the GameWorld and populated during manager initialization.

Note
Uses amortized O(1) lookup via std::unordered_map.
