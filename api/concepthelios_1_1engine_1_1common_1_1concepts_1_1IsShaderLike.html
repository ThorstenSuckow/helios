Detailed Description

Constrains T to types that provide a minimal shader interface.

A type satisfies IsShaderLike if it: 1. Provides `void use()` to bind the shader for rendering. 2. Provides `void applyUniformValues(UniformValueMap&)` to set uniform parameters from a value map.
