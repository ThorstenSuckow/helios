Detailed Description

Data structures for efficient entity management.

This namespace provides core data structures for managing GameObjects, including object pools for memory-efficient entity recycling (GameObjectPool, GameObjectPoolId, GameObjectPoolRegistry), filtered views for querying entities based on state or components (GameObjectFilter, GameObjectView). For coordinated pool/world operations, see GameObjectPoolFacade.
