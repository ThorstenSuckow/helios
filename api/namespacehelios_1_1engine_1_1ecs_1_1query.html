Detailed Description

Entity query utilities for filtering and iterating GameObjects.

This module provides efficient mechanisms for querying GameObjects based on their component composition. It includes GameObjectFilter for filtering by active/enabled state and GameObjectView for lazy component-based iteration with structured binding support.

Defines criteria for filtering GameObjects during iteration.

Used by `GameWorld::find()` and `GameObjectView` to select which entities should be included in the results based on their active state or component state.

Enumerator
None 

No filtering applied.

Active 

Include only active GameObjects.

Inactive 

Include only inactive GameObjects.

ComponentEnabled 

Include GameObjects with enabled components.

ComponentDisabled 

Include GameObjects with disabled components.

All 

Include all GameObjects regardless of state.
