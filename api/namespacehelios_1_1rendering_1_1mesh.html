Detailed Description

Mesh rendering abstractions and command types.

This namespace provides the core types for mesh rendering. MeshRenderable is a high-level component that can be attached to game objects, combining a shared RenderPrototype with optional per-instance MaterialPropertiesOverride. MeshRenderCommand is the low-level command object that encapsulates all data needed for a single mesh draw call.
