Detailed Description

Strongly-typed identifier for font families used in text rendering.

`FontId` provides a compile-time or runtime identifier for font families registered with the `TextRenderer`. It uses FNV-1a hashing to convert string names to unique 32-bit identifiers.

## Usage

```cpp // Compile-time construction from string literal constexpr FontId arialFont("Arial"); constexpr FontId monoFont("Monospace");

// Register font with TextRenderer textRenderer.addFontFamily(arialFont, "fonts/arial.ttf");

// Use in TextMesh TextMesh props{arialFont, {100.0f, 200.0f}, 1.0f}; ```
