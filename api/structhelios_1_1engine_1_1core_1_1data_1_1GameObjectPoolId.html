Detailed Description

Strongly-typed identifier for referencing a GameObjectPool.

GameObjectPoolId provides type-safety when working with multiple object pools. Instead of using raw integers or strings, this wrapper ensures compile-time checking when accessing pools from a registry.

The ID is hashable and comparable, making it suitable for use as a key in associative containers like `std::unordered_map`.

Example usage: ```cpp constexpr GameObjectPoolId BULLET_POOL{1}; constexpr GameObjectPoolId ENEMY_POOL{2};

auto* bullets = registry.pool(BULLET_POOL); auto* enemies = registry.pool(ENEMY_POOL); ```
