Detailed Description

Strongly-typed identifier for SpawnProfile instances.

SpawnProfileId provides type-safe identification for spawn profiles, which define how entities are spawned (pool reference, placer, initializer). Using a dedicated ID type prevents accidental misuse of raw integers and enables compile-time enforcement of correct parameter passing.

The ID supports hashing for use in unordered containers and comparison operators for ordered containers.

## Construction

IDs can be constructed from string literals using FNV-1a hashing: ```cpp constexpr SpawnProfileId ENEMY_PROFILE{"enemy_spawn"}; constexpr SpawnProfileId POWERUP_PROFILE{"powerup_spawn"}; ```

Uninitialized IDs for deferred assignment: ```cpp SpawnProfileId profileId{helios::core::types::no_init}; profileId = SpawnProfileId{"assigned_later"}; ```

## Usage

```cpp std::unordered_map<SpawnProfileId, SpawnProfile> profiles; profiles[ENEMY_PROFILE] = createEnemyProfile(); ```
