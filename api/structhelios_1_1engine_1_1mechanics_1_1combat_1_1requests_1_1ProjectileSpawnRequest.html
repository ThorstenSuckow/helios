Detailed Description

Data transfer object for projectile spawn requests.

ProjectileSpawnRequest contains all parameters needed to spawn a projectile from a source entity. It is created by the ShootCommandDispatcher and queued in the ProjectilePoolManager for deferred processing.

The struct captures:

  • Spawn position (from the source entity)
  • Movement direction (from aim component)
  • Throttle/speed multiplier
  • Inherited velocity (for momentum inheritance)
  • Fire intensity (affects spawn rate accumulation)
