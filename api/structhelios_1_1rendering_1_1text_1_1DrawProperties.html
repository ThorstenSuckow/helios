Detailed Description

Properties defining how text should be drawn on screen.

`DrawProperties` encapsulates the positioning and styling information needed to render text at a specific location with a specific font.

## Properties

  • **fontId:** Identifies which loaded font family to use.
  • **position:** Screen-space coordinates (x, y) for the text origin.
  • **scale:** Multiplier for the text size (1.0 = original size).

## Example

```cpp DrawProperties props{ .fontId = FontId{1}, .position = {100.0f, 200.0f}, .scale = 1.5f }; ```
