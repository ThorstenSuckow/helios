## API Docs (Doxygen)

The `api/` folder contains the full API documentation generated by [Doxygen](https://doxygen.nl). This provides a detailed, class-by-class reference of the entire codebase.

### Generating the Documentation

To generate or update the documentation, run the following command from the project's root directory:

```bash
external/doxygen/doxygen.exe docs/Doxyfile.in
```

### Viewing the Documentation

After generation, you can open the main page here:
[**Open Local API Documentation**](./api/html/index.html)
