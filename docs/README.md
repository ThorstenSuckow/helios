# docs

This directory contains various documentation related to the helios project.

 - [helios coding styleguide](./styleguide.md)
 - [doxygen documentation styleguide](./doxygen-style.md)


## api docs

The `api` folder is populated with documentation generated by [doxygen](https://doxygen.nl) by calling

```bash
   external/doxygen/doxygen.exe docs/Doxyfile.in
```

from the root folder of this project. See [doxygen-style.md](./doxygen-style.md) for more information on the doxygen style used in helios.


Fore more information on the API reference, refer to [heliosapi.md](./heliosapi.md).
