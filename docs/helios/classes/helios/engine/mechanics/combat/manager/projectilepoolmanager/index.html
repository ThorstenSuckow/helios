Description​

Manager for projectile spawning with rate-limited pooling.

ProjectilePoolManager handles the lifecycle of projectile GameObjects. It receives ProjectileSpawnRequests from the ShootCommandDispatcher, queues them, and spawns projectiles during the manager flush phase based on a SpawnCondition.

## Rate Limiting

The manager uses an accumulator-based system for rate limiting:

  • Each request's intensity is multiplied by the spawn budget
  • Fractional spawns accumulate across frames
  • Only whole numbers of projectiles are spawned per frame

This allows smooth fire rate control based on trigger pressure.

## Despawn Handling

The manager also implements PoolRequestHandler to process DespawnRequests, returning projectiles to the pool when they expire or hit something.

spawnProjectiles()​

void helios::engine::mechanics::combat::manager::ProjectilePoolManager::spawnProjectiles (helios::engine::runtime::world::GameWorld & gameWorld, helios::engine::runtime::world::UpdateContext & updateContext)
 inline

Spawns projectiles from queued requests based on spawn budget.

Iterates through all pending spawn requests and spawns projectiles using an accumulator-based rate limiting system:

1. For each request, calculates spawn count as `spawnBudget * intensity` 2. Adds this to the accumulator (preserving fractional values across frames) 3. Spawns floor(accumulator) projectiles and subtracts from accumulator 4. Acquires projectiles from pool and initializes via prepareBullet()

This allows sub-frame spawn timing and smooth fire rate control.

Parameters
gameWorld

The game world for entity access.

updateContext

The current frame's update context.

171 helios::engine::runtime::world::GameWorld& gameWorld,
172 helios::engine::runtime::world::UpdateContext& updateContext
173 ) {
174
175 float spawnBudget = spawnCondition_->spawnBudget(gameWorld, *gameObjectPool_, updateContext);
176
177 for (const auto& spawnRequest : requests_) {
178
179 // add to this gameobjects sourceAccumulator the budget aligned to the intensity.
180 // e.g., if the intensity fot his frame is 0.2, and the spawnBudget is 5,
181 // only 5 * 0.2 = 1 enemies should spawn.
182 // add this fraction to the sourceAccumulator.
183 sourceAccumulator_ += spawnBudget * spawnRequest.intensity;
184
185 // floor the current sourceAccumulator and store in discrete value
186 // so we can iterate
187 size_t sb = static_cast<size_t>(sourceAccumulator_);
188
189 // reduce the sourceAccumulator by this amount, since this is the
190 // exact amount we _want_ to spawn in this frame.
191 sourceAccumulator_ -= static_cast<float>(sb);
192
193 for (size_t i = 0; i < sb; i++) {
194 auto* go = gameObjectPoolFacade_.acquire(gameWorld, *gameObjectPool_);
195 if (!go) {
196 break;
197 }
198 prepareBullet(go, spawnRequest);
199 }
200 }
201 }
