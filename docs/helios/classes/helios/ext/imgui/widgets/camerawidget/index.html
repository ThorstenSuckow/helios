Description​

Debug widget for real-time camera parameter control and visualization.

Provides runtime configuration of camera position, orientation, projection parameters, and view settings through a flat, user-friendly interface. Multiple cameras can be registered and selected via a combo box.

Camera transforms (position, rotation) are controlled via CameraSceneNode, while projection settings (FOV, aspect, near/far) are set directly on the Camera.

info

Not thread-safe. Use from the main/render thread only.
