23 * @class CollisionStateComponent
24 * @brief Stores per-frame collision state for a GameObject.

26 * @details This component is written by the collision detection system each frame
27 * to record whether a collision occurred and provide details about the collision.
28 * Game logic can query this component to react to collisions.
29 *
30 * The state should be reset at the start of each frame by the collision system or
31 * when the owning GameObject is acquired from a pool.
