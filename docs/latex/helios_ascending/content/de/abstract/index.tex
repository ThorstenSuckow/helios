\begin{abstract}
    Wir stellen \textit{helios} vor, eine in C++ entwickelte, ECS-basierte Game Engine zur Umsetzung eines \textit{Geometry Wars}-Klons.\\
    Das Dokument geht auf die Weiterentwicklung des vorangegangenen Prototypen ein und die damit verbundene Konzeption der Kernarchitektur.
    Es analysiert kritisch die Anforderungen, welche die ursprünglichen Architekturentscheidungen infrage gestellt haben, und dokumentiert den Übergang von einem objektorientierten hin zu einem datenorientierten System.
    Dabei werden die Vorteile der neu implementierten Lösung vorangegangenen Zwischenständen gegenübergestellt und innerhalb des ECS-Paradigmas begründet.
\end{abstract}
