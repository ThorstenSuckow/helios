# helios::engine::game::gameplay::combat

Combat-related gameplay systems, components, and commands.

This module provides all building blocks for implementing combat mechanics such as aiming, shooting, and projectile management.

- **components**: Data containers for combat state (aiming direction, shooting cooldowns, weapon configuration).
- **commands**: Input-to-action mapping for combat actions (aim, shoot).
- **systems**: Update logic for projectile pools and firing mechanics.
