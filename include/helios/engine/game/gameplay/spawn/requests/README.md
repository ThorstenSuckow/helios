# helios::engine::game::gameplay::spawn::requests

Data structures for spawn and despawn requests.

This namespace contains lightweight data transfer objects used to communicate spawn and despawn parameters between dispatchers and managers.

## Structures

| Structure | Purpose |
|-----------|---------|
| `SpawnRequest` | Contains pool ID and spawn budget for spawning |
| `DespawnRequest` | Contains entity GUID and optional pool ID for despawning |
