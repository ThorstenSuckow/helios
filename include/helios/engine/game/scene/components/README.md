# helios::engine::game::scene::components

Scene graph integration components.

This module contains components that bridge the game logic layer (GameObjects) with the rendering layer (SceneNodes). These components enable GameObjects to be represented in the scene graph and rendered.

## Components

| Component | Description |
|-----------|-------------|
| `SceneNodeComponent` | Links a GameObject to a SceneNode for rendering. |
