# helios::engine::mechanics::combat::requests

Request types for combat-related deferred operations.

## Overview

This module defines request structures used to communicate between command dispatchers and managers. Requests encapsulate the data needed to perform combat operations during the manager flush phase.

## Key Classes

| Class | Purpose |
|-------|---------|
| `ProjectileSpawnRequest` | Request data for spawning a projectile from a pool |
