# helios::engine::mechanics::match::components

Match state components.

## Components

| Component | Description |
|-----------|-------------|
| `MatchStateComponent` | Stores the current match state on an entity |
| `MatchTimerComponent` | Tracks elapsed time during a match |

## MatchStateComponent

A component that holds a `MatchState` value. Typically attached to a session or global entity to track the current state of the match.

## MatchTimerComponent

Tracks elapsed time during a match. Useful for timing match phases, countdowns, or total match duration.
