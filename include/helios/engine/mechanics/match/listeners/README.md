# helios::engine::mechanics::match::listeners

Listeners for reacting to match state transitions.

## Listeners

| Listener | Description |
|----------|-------------|
| `PlayerSpawnListener` | Activates the player GameObject when entering PlayerSpawn state |

## PlayerSpawnListener

Listens for transitions to the `PlayerSpawn` state and activates the referenced player GameObject. This ensures the player entity becomes active when the match is ready for play.
