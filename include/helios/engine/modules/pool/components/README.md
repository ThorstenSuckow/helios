# helios::engine::modules::pool::components

Components for pool-based entity management.

## Overview

This module provides components that track pool membership for GameObjects. These components enable the pooling system to identify which pool an entity belongs to for proper acquire/release operations.

## Key Classes

| Class | Purpose |
|-------|---------|
| `PoolIdComponent` | Stores the pool ID that owns this GameObject |
