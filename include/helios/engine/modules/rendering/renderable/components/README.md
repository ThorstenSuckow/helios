# helios::engine::modules::rendering::components

Components related to rendering and visual representation.

This module contains components that link game logic entities to the rendering system, enabling GameObjects to have visual representation.

## Components

| Component | Description |
|-----------|-------------|
| `RenderableComponent` | Holds a shared pointer to a Renderable resource for visual display. |
