# helios::engine::modules::ui::widgets

UI widget components and systems.

This namespace provides components for UI widgets such as text displays, along with systems that manage their state and bounds.

## Submodules

| Submodule | Purpose |
|-----------|---------|
| `components/` | Widget state components |
| `systems/` | Systems for updating widget state and bounds |

## Key Classes

| Class | Purpose |
|-------|---------|
| `UiTextComponent` | Text display with template-based value formatting |
| `UiTextBoundsUpdateSystem` | Updates AABB when text content changes |
